FORMATTING
==========

ViewHelpers:
 * ViewHelpers in continuous text should be formatted like that:
	... the <code>cObject</code> ViewHelper
	... there exists a ViewHelper <code>format.date</code>

Code Snippets:
 * Code Snippets are indented with a single TAB character
